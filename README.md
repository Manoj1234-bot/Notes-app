# 📝 Notes App

A modern and responsive **Notes Application** built using **HTML, CSS, and JavaScript**. The application allows users to create, edit, delete, search, pin, export, and import notes with a clean interface. It also supports **Dark/Light Mode**, note color customization, and stores all notes locally using **Local Storage** for offline access.

# ✨ Features

* 📝 Create new notes
* ✏️ Edit existing notes
* 🗑️ Delete notes
* 📌 Pin important notes to the top
* 🎨 Choose different note colors
* 🔍 Search notes instantly
* 🌙 Dark / Light Mode
* 💾 Auto-save using Local Storage
* 📤 Export all notes as JSON backup
* 📥 Import notes from JSON backup
* 📄 Download individual notes as `.txt` files
* 📱 Fully Responsive Design
* ⚡ Fast and lightweight

# ⚙️ How It Works

1. Users can create a new note with a title and description.
2. Each note can be assigned a custom background color.
3. Notes are automatically saved in the browser using Local Storage.
4. Users can edit, pin, search, or delete notes at any time.
5. Notes can be exported as a JSON backup and imported later.
6. Individual notes can be downloaded as `.txt` files.
7. Dark Mode preference is saved for future visits.
